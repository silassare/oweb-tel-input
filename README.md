## O'Web Tel Input

A simple phone input validator and formatter plugin for O'Web.
Depends on libphonenumber by google.

### Installation
```
npm install --save silassare/oweb-tel-input
```

### Build
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
